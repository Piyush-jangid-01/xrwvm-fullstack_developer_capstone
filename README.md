## 📖 Project Description
This is a full-stack web application developed as part of the IBM Full Stack Developer Capstone project. The application allows users to browse car dealerships, view dealer details, read and post reviews, and perform sentiment analysis on reviews.

---

## 🛠️ Tech Stack
- Frontend: React.js, HTML, CSS
- Backend: Django (Python)
- APIs: REST APIs
- Database: SQLite / Cloudant (depending on setup)
- Deployment: IBM Cloud Code Engine (if used)

---

## 🚀 Features
- User Registration and Login
- Dealer Listing and Filtering by State
- Dealer Details and Reviews
- Add Review functionality
- Sentiment Analysis for Reviews
- Admin Panel for Management
